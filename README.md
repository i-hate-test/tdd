## 스터디 방식

- 일정
  - 주 1-2회 완료 목표(2/1~2/28)

- 히스토리
  - 2/1(목) : 주제 및 교재 선정 (테스트 주도 개발 시작하기 최범균 저자)
  - 2/5(토) : 교재 2단원까지 학습 후 스터디 방향 정하기
  - 2/13(화) : 교재 1회독 후 TDD 에 대한 토론
  - 2/22(목) : 각 챕터별 정리 & 테스트 코드 작성 후기
